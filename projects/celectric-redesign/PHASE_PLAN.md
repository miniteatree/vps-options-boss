# Celectric Phase Plan

## Purpose
Work guideline for what to do first and what comes next.

## Phase 0, Preparation
### Goal
Lock direction and scope.

### Tasks
- confirm business direction
- confirm catalog + RFQ model
- confirm blog inclusion
- confirm hosting and storage direction
- confirm client presentation format

### Exit Criteria
- client agrees on direction
- main sitemap is approved
- priority pages are agreed

## Phase 1, Short Term Build
### Goal
Deliver the core public experience.

### Tasks
- homepage
- products listing
- product detail
- brands
- blog / insights
- RFQ flow
- secure document links
- mobile responsive cleanup

### Exit Criteria
- client can browse products and read blog content
- client can request quote
- client can access documents securely

## Phase 2, Mid Term Improve
### Goal
Make the site more useful and searchable.

### Tasks
- better search
- filtering
- related products
- related articles
- resource center
- better SEO landing pages
- more document grouping rules

### Exit Criteria
- users can find products faster
- content supports SEO and internal linking
- document flow is stable

## Phase 3, Long Term Expand
### Goal
Add operational depth.

### Tasks
- Odoo sync
- admin workflow improvements
- CRM integration
- user accounts if needed
- checkout if business model changes

### Exit Criteria
- product data and enquiry flow are connected to operations
- platform can scale without rework

## Working Order
1. confirm scope
2. finalise sitemap
3. define content model
4. build core pages
5. add blog and document flow
6. improve search and integrations
