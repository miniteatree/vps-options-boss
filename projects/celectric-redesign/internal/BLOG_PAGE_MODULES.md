# Celectric Blog / Insights - Page Module Breakdown

## Purpose
Define the recommended modules for the blog listing, category, and article detail pages so the feature can move from planning into implementation.

## Goals
- Support SEO and long-tail discovery
- Build credibility through technical content
- Create clear paths from informational content to enquiry conversion
- Connect content with products, brands, and solutions where relevant

## 1. Blog Listing Page
Suggested URL:
- `/blog`

### Recommended Modules
1. Hero / page intro
   - page title
   - short intro explaining the purpose of the insights section
   - optional featured article highlight

2. Search and filter bar
   - keyword search
   - category filter
   - optional tag filter

3. Featured posts section
   - latest or editor-picked key articles
   - larger card layout

4. Article grid / list
   - featured image
   - title
   - excerpt
   - category
   - publish date
   - author
   - CTA to read more

5. Pagination / load more

6. Newsletter or enquiry CTA block
   - optional depending on business preference
   - can also be replaced with Contact Sales / Request Quote CTA

## 2. Blog Category Page
Suggested URL:
- `/blog/{category}`

### Recommended Modules
1. Category header
   - category name
   - short category description

2. Search / filter controls
   - keep filtering simple to avoid clutter

3. Category article list
   - same card structure as blog listing page

4. CTA block
   - prompt to explore products or contact sales

## 3. Blog Article Detail Page
Suggested URL:
- `/blog/{post-slug}`

### Recommended Modules
1. Article hero
   - title
   - subtitle or short excerpt
   - featured image
   - author
   - publish date
   - updated date where relevant
   - category

2. Article body
   - rich text content
   - images
   - tables where needed for technical comparison
   - callout boxes for key takeaways

3. In-article CTA blocks
   - Request Quote
   - Contact Sales
   - View Related Products

4. Related entities section
   - related products
   - related brands
   - related solutions

5. Related posts section
   - suggested further reading

6. Share / copy link actions
   - optional but useful

7. Bottom CTA
   - clear conversion step after reading

## 4. Homepage Integration
Recommended homepage blog area:
- section title such as Latest Insights or Technical Articles
- 3 to 4 latest featured posts
- link to full blog listing

## 5. UX Guidance
- Keep article cards visually consistent with product and brand cards
- Ensure blog pages support mobile reading well
- Make CTAs helpful, not too aggressive
- Prioritize readability for technical audiences

## 6. Phase Guidance
### Phase 1
- blog listing page
- blog article detail page
- category support
- homepage latest blog section
- basic CTA integration

### Phase 2
- related content automation
- article search improvements
- richer author profiles
- gated downloads or advanced lead magnets if needed
